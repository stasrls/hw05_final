# YaTube

## Описание
Социальная сеть для блогеров.

Вы можете:
- Регистрироваться, восстанавливать пароль по почте
- Создавать, просматривать, редактировать и удалять записи (CRUD)
- Подписываться на других авторов и видеть у себя в ленте их посты
- Загрузка изображений при создании постов
- Комментировать чужие записи
- Создавать свою уникальную страницу и адрес
- Модерировать посты и администрировать пользователей
- Группировать посты

Проект покрыт юнит тестами и прошел код ревью.  

## Запуск проекта в dev-режиме
- Установите и активируйте виртуальное окружение
- Установите зависимости из файла requirements.txt
```
pip install -r requirements.txt
``` 
- В папке с файлом manage.py выполните команду:
```
python3 manage.py runserver
```
